### Types of Drift

| Type | Description | Detection |
|------|-------------|-----------|
| **Data Drift** | Input distribution changes | Statistical tests (KS, PSI) |
| **Concept Drift** | Relationship Xâ†’Y changes | Performance monitoring |
| **Label Drift** | Output distribution changes | Prediction distribution |

### Drift Detection

```python
from scipy import stats

def detect_drift(reference, current, threshold=0.05):
    """Kolmogorov-Smirnov test for drift detection"""
    statistic, p_value = stats.ks_2samp(reference, current)
    
    if p_value < threshold:
        return True, p_value  # Drift detected
    return False, p_value

# Example
reference_data = training_data['feature_1']
current_data = production_data['feature_1']

drift_detected, p_value = detect_drift(reference_data, current_data)
if drift_detected:
    alert("Data drift detected in feature_1!")
```

### ML Pipeline (Kubeflow)

```yaml
# Kubeflow Pipeline Example
@dsl.pipeline(
    name='Training Pipeline',
    description='End-to-end ML pipeline'
)
def ml_pipeline(data_path: str, model_path: str):
    
    # Step 1: Data Preprocessing
    preprocess_op = preprocess_component(
        input_data=data_path
    )
    
    # Step 2: Training
    train_op = train_component(
        train_data=preprocess_op.outputs['train_data'],
        hyperparameters={'lr': 0.01, 'epochs': 100}
    )
    
    # Step 3: Evaluation
    eval_op = evaluate_component(
        model=train_op.outputs['model'],
        test_data=preprocess_op.outputs['test_data']
    )
    
    # Step 4: Conditional Deployment
    with dsl.Condition(eval_op.outputs['accuracy'] > 0.9):
        deploy_component(
            model=train_op.outputs['model'],
            endpoint='production'
        )
```
